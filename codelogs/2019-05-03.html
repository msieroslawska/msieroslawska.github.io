Learned today

isNaN() and Number.isNaN() are two different functions:

isNaN(1); // false
isNaN('1'); // false
isNaN(1.2); // false
isNaN('1x'); // true
Number.isNaN(1); // false
Number.isNaN('1'); // false
Number.isNaN(1.2); // false
Number.isNaN('1x'); // false
